Classes in Python

Classes are used to create a collection of functions and attributes that define objects. They allow us to visualize objects as real-life items with characteristics and behaviors, and also show relationships between objects.

  • How to use classes in Python

    
class Student:
    def __init__(self, name, age, sex):
        self.name = name
        self.age = age
        self.sex = sex

# Creating an object of the class
a = Student("Moses", 21, "Male")
print(a.name)
print(a.age)
print(a.sex)

  • Program to demonstrate instance objects in a class

    
class Fruit:
    def __init__(self, message):
        self.message = message

# Creating instances of the Fruit class
apple = Fruit("Fruits are sweet and nutritious")
banana = Fruit("This instance coexists with others, hence fruits are healthy")

# Accessing and printing instance attributes
print(apple.message)
print(banana.message)

  • Program demonstrating an initializer (__init__) and instance methods

    
class Pet:
    def __init__(self, name, sound):
        self.name = name    # Name of the pet
        self.sound = sound  # Sound the pet makes

    def speak(self):
        print(f"{self.name} {self.sound}")

# Creating instances of Pet
dog = Pet("Dog", "barks")
cat = Pet("Cat", "purrs")

# Calling the speak method
dog.speak()
cat.speak()

    __init__ is the initializer (constructor) that runs when an object is created. self refers to the current instance and allows access to attributes of that object. speak() is a method that outputs pet details. The f"{self.name} {self.sound}" is an f-string, a clean way to format output.

  • A program to show the basic working of a money calculator

    
class BankAccount:
    def __init__(self, balance=0):
        self.balance = balance  # Initial balance, default is 0

    def deposit(self, amount):
        self.balance += amount
        return self.balance

    def withdraw(self, amount):
        if amount > self.balance:
            return "Insufficient balance"
        self.balance -= amount
        return self.balance

# Example usage
account = BankAccount(1000)  # Starting with 1000
print("After deposit:", account.deposit(1000000))
print("After withdrawal:", account.withdraw(500000))
