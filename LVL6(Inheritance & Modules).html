Inheritance and Modules in Python

üêæ Inheritance in Python

Inheritance in Python is used to pass class attributes onto another subclass. The subclass gains all the traits of the parent class, but it can also have its own attributes, which overwrite or extend the parent‚Äôs code.

üëâ Example: Create a class Animals, then create a Mammals subclass. A Dog inherits all the attributes of Mammals (which itself inherits from Animals).

Python Inheritance Example


# Base class
class Animals:
    def __init__(self):
        # Attributes for all animals
        self.breathing_system = "All animals have a respiratory system that uses oxygen"
        self.digestion_system = "All animals digest food to get energy for survival"
        self.locomotive_system = "All animals have a way of moving from one place to another"

# Subclass that inherits from Animals
class Mammals(Animals):
    def respiratory(self):
        # Overwrites or extends parent behavior
        print("Most mammals have a set of lungs and a nose hence:", self.breathing_system)

    def digest(self):
        print("Most mammals have a digestive tract with teeth and chemicals hence:", self.digestion_system)

    def movement(self):
        print("All mammals can move by the use of their four legs hence:", self.locomotive_system)

# Another subclass that inherits from Mammals
class Dog(Mammals):
    def inherited_attributes(self):
        print("These are the general characteristics of a dog")

# Example Usage
a = Mammals().respiratory()
b = Mammals().digest()
c = Mammals().movement()
d = Dog()
d.inherited_attributes()

üì¶ Modules in Python

A module in Python is simply a file that contains Python code (functions, classes, or variables). Modules help organize code into separate files, making it reusable and easier to maintain.

üëâ You can use: - Built-in modules (like math, os, random) - Custom modules (your own .py files)

Custom Module Example

File: mymodule.py


# mymodule.py
# A simple custom module

def greeting(name):
    return f"Hello, {name}! Welcome to Python modules."

def add_numbers(a, b):
    return a + b

File: main.py


# main.py
import mymodule   # importing the custom module

# Using functions from the module
print(mymodule.greeting("Moses"))
print("The sum is:", mymodule.add_numbers(5, 7))
